# 개인별 청구 현황 (PERSON_DATA / PERSON_DAILY) 계산 로직

> `update_all.py` Step 6~6-2 (145~196줄)
> 원본: `2026 data RAW_Claim.xlsx` — sheet_name=2

---

## 1. 원본 컬럼 구조

| 인덱스 | 컬럼 | 용도 |
|--------|------|------|
| cols[1] | 상태 | '청구' 또는 '취소' 포함 필터 |
| cols[8] | Task | 개인별 집계 시 Task=1만 |
| cols[13] | 날짜 (N) | YY-MM 월 / YY-MM-DD 일 추출 |
| cols[22] | 금액 | 금액 합산 |
| cols[23] | 담당자 ID | 개인 식별 |

---

## 2. PERSON_DATA 집계 기준

| 항목 | 기준 |
|------|------|
| 필터 | 상태에 '청구' 또는 '취소' 포함 |
| Task 필터 | **Task=1만** (개인별 집계) |
| 집계 단위 | 담당자 + 월 + ClaimType + 상태 |
| 집계 항목 | count (건수), amount (금액합산) |

### 전체합계 행 추가
- `담당자='전체합계'` 로 Task 필터 없이 전체 집계 추가
- 보증매출(BRANCH_DATA)과 동일 기준

---

## 3. PERSON_DATA 구조

```
[ 담당자ID, 월, ClaimType, 상태, 건수, 금액 ]

예) ["홍길동", "26-01", "BSI", "청구", 5, 12000000]
    ["전체합계", "26-01", "BSI", "청구", 174, 78961624]
```

---

## 4. PERSON_DAILY 집계 기준 (일별 현황)

| 항목 | 기준 |
|------|------|
| 필터 | 상태='청구' + 당월(오늘 기준 YY-MM)만 |
| Task 필터 | **Task=1만** |
| 집계 단위 | 담당자 + 일자(YY-MM-DD) + ClaimType |
| 집계 항목 | count (건수), amount (금액합산) |

---

## 5. PERSON_DAILY 구조

```
[ 담당자ID, 날짜, ClaimType, 건수, 금액 ]

예) ["홍길동", "26-03-15", "BSI", 2, 5000000]
```

---

## 6. 대시보드 표시

- 월별 개인 실적 테이블
- 당월 일별 추이 차트
- 전체합계와 개인 비교
