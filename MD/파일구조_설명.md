## 파일 2: `2026_STAGE2,3 Whole sale data_ Total.xlsx`

### 역할
파일1의 `클레임현황` 시트를 외부 참조(`[1]클레임현황`)하여 월별/유형별 집계를 보여주는 대시보드 소스 파일.

### 시트 구성 (3개, 구조 동일)

| 시트명 | 집계 금액 |
|--------|---------|
| `클레임 PART only_RAW(보증마감기준)` | J열(부품금액) 기준 |
| `클레임 PART only_RAW(회계마감기준)` | J열(부품금액) 기준 |
| `클레임 청구 볼륨_RAW` | M열(총클레임금액) / R열(총승인금액) 기준 |

### 각 시트 행 구조 (50행 × 16열)

```
행4:  헤더 (유형/월 | Claim Type | Total | 25-01 | 25-02 | ... | 25-12)
행5~10:  [2025] 청구 - Claim Type별 × 월별 합계
행11:    [2025] 청구 Total
행12~17: [2025] 승인
행18:    [2025] 승인 Total
행19~24: [2025] 승인대기
행25:    [2025] 승인대기 Total

행29: 헤더 (유형/월 | Claim Type | Total | 26-01 | ... | 26-12)
행30~35: [2026] 청구
행36:    [2026] 청구 Total
행37~42: [2026] 승인
행43:    [2026] 승인 Total
행44~49: [2026] 승인대기
행50:    [2026] 승인대기 Total
```

### SUMIFS 참조 관계
```
파일2 셀 = SUMIFS(
    파일1.클레임현황!$M:$M,        ← 집계할 금액열 (M=총클레임 or J=부품 or R=총승인)
    파일1.클레임현황!$T:$T, C5,    ← Claim Type 필터
    파일1.클레임현황!$S:$S, "25-01" ← 월(YY-MM) 필터
)
```

