## MD 파일 작성 규칙

### 파일 네이밍
```
번호_카테고리명_데이터명.md

예)
01_보증매출_BRANCH_DATA.md
06_클레임볼륨_WHOLESALE_DATA.md
결과/2026_Q1_TC미실시율.md
```

### 새 MD 파일 만들 때
1. 두 경로에 동시 저장
2. 로직 문서 → 번호 붙여서 루트에
3. 결과값 스냅샷 → `결과/` 하위 폴더에
