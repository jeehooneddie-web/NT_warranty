# 고품 QR 스캐너

> `index.html` — view-qr-scanner
> 접근 권한: 관리자 전용 (admin-only)

---

## 1. 개요

모바일 카메라로 BMW 고품 라벨 QR 코드를 스캔하여 부품 정보를 확인하고 Google Sheets에 실시간 저장하는 기능.

---

## 2. 기술 스택

| 항목 | 내용 |
|------|------|
| QR 인식 라이브러리 | html5-qrcode@2.3.8 (CDN) |
| 데이터 저장 | localStorage (당일 목록) + Google Apps Script 웹훅 |
| 접근 방식 | 모바일 크롬 후면 카메라 (facingMode: environment) |

---

## 5. 스캔 흐름

```
카메라 시작
  └→ html5QrScanner.start() — fps:10, qrbox:250×250
        └→ QR 인식 시 pause(true) 호출 (중복 콜백 차단)
              └→ qr-reader 영역 숨김
              └→ handleQRResult(decodedText) 호출
                    └→ CSV 파싱 → 결과 박스 표시
                    └→ 저장 버튼 클릭
                          └→ localStorage 저장
                          └→ Google Sheets POST
                          └→ 카메라 자동 재시작 (resume())
```

---

## 6. Google Sheets 연동

- **웹훅 URL**: `QR_WEBHOOK_DEFAULT` (index.html 상수)
- **전송 방식**: fetch POST, mode: no-cors
- **저장 위치**: Google Apps Script 웹앱 → 활성 시트에 행 추가
- **저장 필드 (시트 컬럼 순서)**:
  스캔시간(0), 딜러번호(1), 지점(2), 클레임번호(3), 차대번호(4), 모델(5),
  디펙트코드(6), 주행거리(7), 등록일(8), 수리일(9), 부품번호(10), 부품명(11),
  Part순서(12), 전체Part수(13), UnitNo(14), 보증타입(15), 보증단계(16)
- **데이터 읽기**: GET은 `QR_CSV_URL` 공개 CSV로 대체 (CORS 우회)

---

## 7. 로컬 저장 (localStorage)

- **키**: `qr_scans_today`
- **구조**: JSON 배열, 스캔 객체 누적
- **용도**: 당일 스캔 목록 화면 표시 + 중복 스캔 감지

---

## 8. 주요 함수

| 함수 | 역할 |
|------|------|
| `initQRScanner()` | 뷰 진입 시 초기화 (웹훅 상태, 스캔 목록) |
| `startQRCamera()` | html5-qrcode 스캔 시작 |
| `stopQRCamera()` | 카메라 완전 정지 (stop + DOM 정리) |
| `handleQRResult(text)` | QR 원문 파싱 → 결과 박스 렌더링 |
| `submitQRScan()` | localStorage 저장 + Sheets 전송 + 카메라 재시작 |
| `resetQRResult()` | 결과 박스 초기화 + 카메라 resume |
| `renderQRScanList()` | 당일 스캔 목록 렌더링 |
| `clearTodayScans()` | 당일 목록 초기화 |
